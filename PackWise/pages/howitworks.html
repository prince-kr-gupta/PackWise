How IT Works

1️⃣ Home Page (index.html)

User lands on the home page

Reads about the app

Clicks "Start Planning" to go to the Planner

2️⃣ Trip Planner (planner.html)

User fills in the form:

Destination, Days, Weather, Trip Type, Travelers, Budget

On submit, all data is saved as one object in localStorage under the key "trip" User is redirected to the Checklist page

3️⃣ Packing Checklist (checklist.html)

Page loads and reads trip data from localStorage

Displays the Trip Summary at the top

Shows a smart packing list based on the trip details

User checks off items as they pack

Clicking "Save Checklist" saves progress to localStorage under key "packwiseChecklist"

User can go to Budget or Dashboard from here

4️⃣ Budget Tracker (budget.html)

Auto-loads the budget amount from localStorage (trip.budget)

User adds expenses with a category, description and amount

Each expense is saved to localStorage under key "packwiseBudget"

Running totals (Spent, Remaining) update instantly

Red warning appears if spending exceeds budget

5️⃣ Dashboard (dashboard.html)

Reads all 3 data sources from localStorage:

"trip" → Trip Summary

"packwiseChecklist" → Packing Progress

"packwiseBudget" → Budget Overview

Displays everything in one clean view

User can click "Export as PDF" to print/save the dashboard
